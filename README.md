Копирования репозитория с последующим копирвоанием движка
```bash
git clone https://github.com/Cinkafox/RobustExperiment
cd RobustExperiment
git submodule update --init --recursive
```

Сборка и запуск
```bash
dotnet build
dotnet run --project Content.Client
```
